## Sistema de Login de Funcionários

Este projeto contém uma interface simples com HTML, CSS e JavaScript para login de funcionários.  
Após o login bem-sucedido, o usuário é redirecionado para a página tickets.html.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Estrutura de arquivos
- index.html — Página de login
- style.css — Estilos da interface
- login.js — Lógica do formulário
- tickets.html — Página interna após login

## Como usar
1. Baixe ou clone o repositório.
2. Abra o arquivo index.html no navegador.
3. Faça login usando:
   - Usuário: *admin*
   - Senha: *1234*
